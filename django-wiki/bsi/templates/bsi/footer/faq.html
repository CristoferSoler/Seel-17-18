Frequently Asked Questions



What is ISAM?

The ISAM (Information Security Advice Manual) is a IT security platform interface to IT – Basic Protection Compendium of the Federal Office for Information Security (BSI). The platform consist of BSI Pages (see 3), user articles (see 5), virtual assistant (see 8) and Information page (see 9). The purpose of the platform is to help the users to find a solutions to IT security problems.

What is BSI IT Basic Protection Compendium?

The Federal Office for Information Security of Germany (BSI) publishes a foundation paper for IT security called IT Basic Protection Compendium. The first part was published in 2005 and aimes to explains the general principles of in, as well as raising awareness of standards in this sector. The topic is rounded off by a collection of threats and measures.

What are the components in BSI IT BP Compendium?

The IT BP Compendium of the BSI contains the components that cover specific topics with specific IT threats and corresponding protective measures.

What are BSI Pages? Can I edit the BSI Pages?

The BSI pages are articles, written and reviewed by trained specialists, therefore there is no need to edit these topics. For adding further detailed information regarding a specific topic it is advised to write a user article and link it to the proper BSI page.

What are User Articles?

User Articles are articles written by users.The articles are based on their own knowledge, experience and eventuel appropriate to the country in which this platform is operated.

How to create new user articles?

After registering and logging in you are a user and have the possibility to write user articles. Clicking the link “User articles” in the header you reach the user area,by clicking on the (+) sign you can create a new article that you can modify. There are 2 steps for creating a new article, first one is writing the content of the article (title and contents). The second step offers you the possibility to link the article to BSI pages.

How to link articles?

A user has the possibility to link a User topic to a BSI page. There are two ways to do so:
  • Either after creating a new topic: in the second part of creating a topic, the user can choose a BSI page from the tree view or search for a topic in a search bar.
  • Through clicking the “Edit” button at an existing user generated topic, you can choose a BSI page from the Table of content on the left side.

What is the Virtual Assistant?

The “Virtual Assistant” is a tool that attempts to determine which IT security problem the user has by asking them a series of questions. For detailed information click here.

Can I ask the community questions about specific topics, problems?

On the top right of this site you can find the Information Page. There you have the possibility to ask general questions including tips about using the platform, or questions about a technical topic you didn’t find through the search in the articles.

Who is moderating the content of this platform?

The content of this platform moderating administrators and moderators.

How to become a Moderator?

An active user can become a moderator by getting promoted an admin.

What is the archive? Can I edit articles in the archive?

In the archive you find user articles and older versions of the IT Basic Protection Compendium that are not up to date anymore and that have been replaced with newer articles. You cannot edit any article in the archive.

Why was my article moved to the archive?

Your article was linked to a BSI page in the old version of The IT BP Compendium, which means the content of your article may not be up-to-date, therefore it is now visible only in the archive.

Which languages can be used to write articles?

You can write the articles only in english.
